# Deep Learning
Follow the instructions below carefully


1. Set up
    1.1. Find Path
        - Find the path of Gym
        - If you have followed the attached install well, it is usually in the path of Python(..\Python\Python37\Lib\site-packages\gym)
        
    1.2. Register gym environment
        - Enter the envs folder
        - Unzip envs.zip from the current folder location
        - The current step is to register the already created Gym environment

    1.3 Check Running
        1.4.1. Run
            - python -m spinup.run ppo --env SnakeLocomotion-v0 --exp_name test1 --epochs 5
               
        1.4.2. Result
            - python -m spinup.run plot data/test/test_s0
            - python -m spinup.run test_policy data/test/test_s0
